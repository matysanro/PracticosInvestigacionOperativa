💡 Problema 1: Casas

¿Cómo construir casas para ganar más?

🔧 Datos

  • Tipo A: €300.000 costo, €40.000 ganancia
  • Tipo B: €200.000 costo, €30.000 ganancia
  • Presupuesto: €18.000.000
  • Máximo: 80 casas

📏 Modelo

  • Max 40.000x + 30.000y
  • 3x + 2y ≤ 180
  • x + y ≤ 80

📊 Solución

  • ✅ 20 casas tipo A
  • ✅ 60 casas tipo B
  • 💰 Beneficio: €2.600.000

💻 Código Python


from scipy.optimize import linprog
c = [-40000, -30000]
A = [[3, 2], [1, 1]]
b = [180, 80]
res = linprog(c, A_ub=A, b_ub=b, bounds=[(0,None),(0,None)], method='highs')
print(res)

📘 Problema 2: Edición de Revistas

Maximizar ingresos usando tinta limitada

🔧 Datos

  • Deportiva: 2 negros, 1 color, €3
  • Cultural: 2 negros, 2 color, €5
  • Cartuchos: 500 negros y 500 color

📏 Modelo

  • Función: Max 3x + 5y
  • x + y ≤ 250
  • x + 2y ≤ 500

📊 Solución

  • ✅ 0 deportivas
  • ✅ 250 culturales
  • 💸 Ingreso: €1.250

💻 Código Python


from scipy.optimize import linprog
c = [-3, -5]
A = [[1, 1], [1, 2]]
b = [250, 500]
res = linprog(c, A_ub=A, b_ub=b, bounds=[(0,None),(0,None)], method='highs')
print(res)
