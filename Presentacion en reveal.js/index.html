    📚 Problema 1: Edición de Revistas

    Maximizar ingresos con recursos limitados

    🔧 Planteo

    • Revista Deportiva: 2 negros, 1 color, €3
    • Revista Cultural: 2 negros, 2 color, €5
    • Cartuchos disponibles: 500 negros y 500 color

    📏 Modelo Matemático

    • Variables: x = deportivas, y = culturales
    • Función Objetivo: Max 3x + 5y
    • Restricciones:
      • x + y ≤ 250
      • x + 2y ≤ 500
      • x, y ≥ 0

    📊 Resultado

    • ✅ 0 deportivas
    • ✅ 250 culturales
    • 💸 Ingreso máximo: €1.250

    🧪 Código Python

    
from scipy.optimize import linprog

c = [-3, -5]
A = [[1, 1], [1, 2]]
b = [250, 500]
x_bounds = (0, None)
y_bounds = (0, None)

res = linprog(c, A_ub=A, b_ub=b, bounds=[x_bounds, y_bounds], method="highs")

if res.success:
    print("=== RESULTADO ÓPTIMO ===")
    print(f"Deportivas: {res.x[0]:.2f}")
    print(f"Culturales: {res.x[1]:.2f}")
    print(f"Ingreso: €{-res.fun:.2f}")

    🏠 Problema 2: Construcción de Casas

    Maximizar beneficios respetando el presupuesto

    🔧 Datos

    • Tipo A: €300.000, beneficio €40.000
    • Tipo B: €200.000, beneficio €30.000
    • Presupuesto: €18.000.000
    • Máx. casas: 80

    📏 Modelo Matemático

    • Variables: x = casas A, y = casas B
    • Función Objetivo: Max 40.000x + 30.000y
    • Restricciones:
      • 3x + 2y ≤ 180
      • x + y ≤ 80
      • x, y ≥ 0

    📊 Resultado

    • ✅ 20 casas A
    • ✅ 60 casas B
    • 💰 Beneficio: €2.600.000

    🏗️ Código Python

    
from scipy.optimize import linprog

c = [-40000, -30000]
A = [[3, 2], [1, 1]]
b = [180, 80]
x_bounds = (0, None)
y_bounds = (0, None)

res = linprog(c, A_ub=A, b_ub=b, bounds=[x_bounds, y_bounds], method="highs")

if res.success:
    print("=== RESULTADO ÓPTIMO ===")
    print(f"Tipo A: {res.x[0]:.2f}")
    print(f"Tipo B: {res.x[1]:.2f}")
    print(f"Beneficio: €{-res.fun:.2f}")
